#upload
1.git下拉檔案到本機
2.確認下載目錄中 \drinkmaster\ 下.gitignore 內容是空的 可以用nopad++開啟
3.eclipse 設定workspace JSP HTML CSS contentype 為UTF-8 
4.設定maven路徑
5.匯入剛剛下載的專案內容 匯入->maven ->exist maven project
6.測試再將內容傳回自己git
